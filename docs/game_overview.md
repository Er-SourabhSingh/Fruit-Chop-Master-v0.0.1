# 🎮 Fruit Ninja Desktop Game

## Overview
A fast-paced arcade slicing game where the player cuts fruits using mouse movement.

## Objective
- Slice maximum fruits
- Avoid bombs
- Achieve high score

## Platform
- Desktop (Windows)

## Input
- Mouse drag (slice action)

## Core Idea
Fruits are thrown upward and fall down. Player slices them before they fall.

## Game End Conditions
- 3 fruits missed
- Bomb sliced
